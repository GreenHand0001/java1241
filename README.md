# 校园管理系统简介 java1241

## 项目概述

校园管理系统是一款基于Spring Boot与Vue.js技术栈，旨在为校园环境提供高效管理解决方案的系统。该系统采用了前后端不分离的架构模式，确保了操作的流畅性和数据的安全性。

## 技术栈

- 开发环境：Windows 10
- 架构模式：MVC/前后端分离
- JDK版本：Java JDK 1.8
- 开发工具：IDEA
- 数据库版本：MySQL 8.0
- 数据库可视化工具：Navicat
- 服务器：Spring Boot 自带 Apache Tomcat
- 主要技术：Java、Spring Boot、MyBatis、MySQL、JQuery、HTML、Vue.js
